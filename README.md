# README for Depot App

git clone this repository

### Local
```
bundle install
rake db:setup
rake db:seed
rails s
```

###Docker
* install Docker
* install Docker-Compose

```
docker-compose up
```
###Heroku
* install Heroku-CLI

```
heroku create
git push heroku master
```
